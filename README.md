## Intervalo de Números

Dado uma lista de números inteiros, agrupe a lista em um conjunto de intervalos

Exemplo:

Entrada: 100, 101, 102, 103, 104, 105, 110, 111, 113, 114, 115, 150

Saída: [100-105], [110-111], [113-115], [150]

    
## Rodar a aplicação

clonar projeto = https://github.com/leogcjr/intervalo-de-numeros.git

Executar classe main Intervalo.java

- pela linha de comando executar o comando abaixo diretamente na pasta da classe

1º javac Intervalo.java

2º java Intervalo.class

- Caso preferir executar pela IDE - Run As > JavaAplication
